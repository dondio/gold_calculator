راهنمای سریع انتشار اپ محاسبه‌گر خرید طلا (برای GitHub Pages)

1) اگر اکانت گیت‌هاب ندارید، در https://github.com ثبت‌نام کنید.
2) یک ریپازیترِ جدید بسازید؛ نام پیشنهادی: gold-calculator-pwa (Public)
3) فایل‌های این بسته را از پوشه‌ی پروژه در ریپو آپلود کنید (Add file → Upload files) یا از Git استفاده کنید.
   فایل‌های مهم: index.html, app.js, sw.js, manifest.json, icons/*
4) به Settings → Pages بروید. Source را روی Branch: main و Folder: / (root) قرار دهید و Save کنید.
5) بعد از چند دقیقه، سایت شما در آدرس https://<username>.github.io/<repo>/ در دسترس خواهد بود.
6) دقت: برای کارکرد کامل Service Worker و PWA مطمئن شوید که آدرس با HTTPS لود می‌شود (GitHub Pages HTTPS را فعال می‌کند).
7) نصب در iOS: با Safari باز کنید → Share → Add to Home Screen. (iOS به اعلان نصب خودکار پاسخ نمی‌دهد).

نکات توسعه‌ای که می‌توانید اضافه کنید:
- تبدیل عیار (مثلاً 24 به 18) و محاسبه بر اساس عیار.
- امکان ذخیره نرخ لحظه‌ای از API قیمت طلا (نیاز به کلید یا یک سرویس آزاد).
- افزودن گزینه برای سود ثابت به جای درصد.
- افزودن زبان انگلیسی/دو زبانه.

اگر مایل باشی، من می‌تونم قدم‌به‌قدم توی همین گفتگو کمکت کنم تا این فایل‌ها رو به ریپوی گیت‌هاب آپلود کنی و Pages رو فعال کنیم.
